# Gregorio Jaca Portfolio

A minimalist, single-page personal portfolio built with Astro and Tailwind CSS.

## Features
- **Dynamic Background**: Real-time percolation theory visualization using HTML5 Canvas.
- **GitHub Integration**: Live contribution graph using `github-calendar`.
- **Responsive Design**: Mobile-friendly layout with a centered content column.
- **Global Config**: Centralized layout and parameter management in `src/config.ts`.

## Tech Stack
- [Astro](https://astro.build/)
- [Tailwind CSS](https://tailwindcss.com/)
- [GitHub Pages](https://pages.github.com/)

## Development
```bash
npm install
npm run dev
```

## Deployment
Automatically deployed via GitHub Actions on every push to `main`.
```bash
npm run build
```

## Prototypes
Check the `prototypes/` directory for standalone HTML visualizations of percolation theory algorithms.
